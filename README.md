
---------------
## Controls:
- `ESC` - interrupts program execution
- `WASD` - movement
- `F1` - open/close ImGui
- `B` - Blinn-Phong on/off
- `H` - HDR on/off
- `SPACE` - bloom on-off
- `DOWN` - decrease exposure
- `UP` - increase exposure
- `Q` - decrease heightscale
- `E` - increase heightscale
- `G` - greyscale
